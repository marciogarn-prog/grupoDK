Imagens da página inicial (opcional)

Para usar fotos do Instagram no lugar das artes SVG de exemplo:

1. Salve suas imagens nesta pasta com estes nomes:
   - centro-hero.jpg   (ou .webp) — DK Centro Automotivo
   - locadora-hero.jpg (ou .webp) — DK Locadora

2. Em index.html, troque os atributos src das tags <img> na seção inicial
   de "*.svg" para "*.jpg" (ou o formato que você usar).

Proporção sugerida: largura ~1200px ou mais, formato paisagem.
